# AI Masterclass Demo

An interactive single-page web application built using **HTML**, **CSS**, and **JavaScript** to demonstrate real-world AI concepts in a simple and engaging way.

## Features

- Explain why professionals should learn AI
- Role-based AI use cases for:
  - Python Developers
  - DevOps Engineers
  - Embedded Engineers
  - Software Engineers
- Live AI workflow visualization
- Agentic AI routing demonstration
  - Master Agent
  - Sub-Agent
  - Tool execution flow

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Running the Project

### Option 1: Open Directly

Simply open `index.html` in your preferred web browser.

### Option 2: Run a Local Server

```bash
python -m http.server 8000
```

Then open:

```
http://localhost:8000/
```

## Purpose

This project is designed for AI workshops and masterclasses to help learners understand:

- How AI fits into different engineering roles
- How an AI workflow executes step by step
- The concept of Agentic AI using a Master Agent and specialized Sub-Agents
- Practical AI applications through interactive demonstrations
