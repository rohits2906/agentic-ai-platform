# Lite Website Project (Native HTML)

This is the Lite-mode scaffold: a native HTML/CSS/JS static project published
directly from source files.

- Edit `index.html` for structure and copy.
- Reuse `assets/styles.css` variables, layout helpers, and component classes
  before adding one-off styles.
- Put shared JavaScript in `assets/main.js`; keep inline scripts for tiny
  one-page interactions only.
- Keep resource references relative to the project root and commit all assets
  needed by the page.
- Publish with `build_and_deploy_html_project`.
